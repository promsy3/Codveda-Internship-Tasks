# Codveda Data Analytics Internship - Level 1

This repository contains the tasks completed during my Data Analytics internship at **Codveda**. The focus of Level 1 is on foundational data cleaning and exploratory data analysis (EDA).

## 🚀 Projects Overview

### Task 1: Data Cleaning and Preprocessing
- **Goal:** Prepare raw data for analysis.
- **Key Actions:**
  - Handled missing values and removed duplicates.
  - Standardized data formats for consistency.
  - Assigned proper column headers to the Boston Housing and Iris datasets.

### Task 2: Exploratory Data Analysis (EDA)
- **Goal:** Identify patterns, trends, and summary statistics within the Iris Dataset.
- **Key Actions:**
  - Calculated **Summary Statistics** (Mean, Median, Mode, Std Dev).
  - Visualized data distributions using **Histograms** and **Boxplots**.
  - Analyzed feature relationships using a **Correlation Heatmap**.
  - Identified distinct species clusters using **Scatter Plots**.

## 🛠️ Tools & Libraries Used
- **Language:** Python
- **Libraries:** Pandas, NumPy, Matplotlib, Seaborn
- **Environment:** Jupyter Notebook / Google Colab

## 📈 Key Insights
- The **Setosa** species forms a highly distinct cluster compared to Versicolor and Virginica.
- There is a strong positive correlation between **Petal Length** and **Petal Width**.
- Data cleaning ensured that no outliers or null values skewed the statistical summary.
