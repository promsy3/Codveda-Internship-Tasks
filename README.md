# Codveda Data Analytics Internship Portfolio 🚀

This repository documents my comprehensive journey through the **Codveda Data Analytics Virtual Internship**. It showcases a progression from foundational data processing to advanced machine learning and interactive business intelligence.

---

## 📁 Level 1: Data Foundations & EDA
*Focus: Data integrity and initial discovery.*

### Task 1: Data Cleaning & Preprocessing
- **Objective:** Prepare raw datasets for reliable analysis.
- **Actions:** Handled missing values, removed duplicates, and standardized headers for the Boston Housing and Iris datasets.

### Task 2: Exploratory Data Analysis (EDA)
- **Objective:** Uncover underlying patterns and statistical trends.
- **Actions:** Generated summary statistics and visualized distributions using **Histograms** and **Boxplots**. Used **Correlation Heatmaps** to identify feature relationships.

---

## 📈 Level 2: Intermediate Machine Learning
*Focus: Predictive modeling and unsupervised learning.*

### Task 1: Regression Analysis
- **Objective:** Predict numerical values using linear relationships.
- **Actions:** Built a **Linear Regression** model to predict house prices based on room counts, achieving a validated R-squared score.

### Task 3: Clustering Analysis
- **Objective:** Group data points based on mathematical similarity.
- **Actions:** Implemented **K-Means Clustering** on the Iris dataset. Used the **Elbow Method** to determine the optimal number of clusters ($k=3$).

---

## 🏆 Level 3: Specialized Insights & BI
*Focus: Advanced modeling and interactive storytelling.*

### Task 1: Advanced Predictive Modeling
- **Objective:** Build a high-accuracy classification system.
- **Actions:** Developed a **Random Forest Classifier**. Employed **Hyperparameter Tuning (GridSearchCV)** to optimize model performance, evaluated via Confusion Matrices and Classification Reports.

### Task 2: Interactive Dashboard Development
- **Objective:** Create a user-driven data exploration tool.
- **Actions:** Designed a dynamic dashboard in **Tableau Public**. Integrated **interactive slicers and filters** to allow real-time exploration of housing market trends.

---
