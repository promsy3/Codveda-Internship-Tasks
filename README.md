# Codveda Data Analytics Internship

This repository contains my progress and completed projects for the Data Analytics Internship at **Codveda**. The internship is structured into levels of increasing complexity, focusing on real-world datasets and advanced analytical techniques.

---

## 📁 Level 1: Data Foundations & EDA
*Goal: Mastering data cleaning and exploratory analysis.*

### Task 1: Data Cleaning and Preprocessing
- **Objective:** Prepare raw data for analysis.
- **Key Actions:** Handled missing values, removed duplicates, and assigned proper headers to the Boston Housing and Iris datasets.

### Task 2: Exploratory Data Analysis (EDA)
- **Objective:** Uncover patterns and trends.
- **Key Actions:**
  - Generated **Summary Statistics** (Mean, Median, Mode, Std Dev).
  - Visualized distributions using **Histograms** and **Boxplots**.
  - Analyzed feature relationships with a **Correlation Heatmap**.

---

## 📈 Level 2: Advanced Analysis & Machine Learning
*Goal: Applying predictive modeling and unsupervised learning.*

### Task 1: Regression Analysis
- **Objective:** Predict numerical outcomes based on data features.
- **Key Actions:** Built a **Linear Regression** model to predict house prices based on room counts. Validated the model using **R-squared** and **MSE** metrics.

### Task 3: Clustering Analysis
- **Objective:** Group data points by similarity without predefined labels.
- **Key Actions:**
  - Implemented **K-Means Clustering** on the Iris dataset.
  - Used the **Elbow Method** to mathematically determine the optimal cluster count (k=3).
  - Visualized multi-dimensional data in 2D cluster plots.
